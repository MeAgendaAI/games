# 🎮 World_TOP - Voxel Game

Um jogo voxel estilo Minecraft melhorado, desenvolvido com Three.js, com dinossauros montáveis, sistema de crafting e muito mais!

## 🚀 Demo

Acesse: [https://meagendaai.github.io/games/](https://meagendaai.github.io/games/)

## ✨ Características

### 🦖 Dinossauros Montáveis e Voadores
- Encontre dinossauros verdes espalhados pelo mundo
- Clique neles de mão vazia para montar
- Use **WASD** para controlar e **Espaço** para voar!
- Pressione **R** para desmontar

### 🎒 Sistema de Items e Drops
- Items caem no chão quando você mata criaturas
- Coleta automática ao passar perto
- Sistema de inventário completo com crafting

### 🏃 Movimento Melhorado
- Velocidade de caminhada aumentada
- Corrida com Shift
- Voo livre (duplo Espaço)
- Física realista

### 🌍 Mundo Procedural
- Geração de terreno infinito
- Biomas variados (planície, deserto, montanha)
- Rios serpenteantes
- Árvores e vegetação
- Casas com rádio Login FM!

### 🔨 Crafting e Construção
- Quebre e coloque blocos
- Sistema de crafting para criar ferramentas
- Espadas de madeira e pedra
- Comida para recuperar vida

## 🎮 Controles

### PC
- **WASD** - Mover
- **Mouse** - Olhar
- **Espaço** - Pular (duplo para voar)
- **Shift** - Correr / Descer (voando)
- **Clique Esquerdo** - Colocar bloco
- **Clique Direito** - Quebrar bloco / Montar dinossauro
- **Scroll** - Trocar slot
- **1-9** - Selecionar slot
- **E** - Inventário
- **F** - Comer fruta selecionada
- **R** - Desmontar do dinossauro
- **[ ]** - Volume da rádio/telão

### Mobile
- **Joystick virtual** - Movimento
- **Arrastar tela** - Olhar
- **Botões na tela** - Ações (pular, quebrar, colocar, inventário)

## 🐄 Criaturas

- **Vacas** 🐄 - Passivas, dropam carne quando mortas
- **Zumbis** 🧟 - Hostis, atacam à noite
- **Dinossauros** 🦖 - Montáveis e voadores!

## 📻 Rádio Login FM

Encontre casas pelo mundo com TVs que tocam a Login FM ao vivo! Também há um telão gigante perto do spawn.

## 🛠️ Tecnologias

- Three.js (WebGL)
- Vanilla JavaScript
- HTML5 Canvas para texturas dinâmicas
- Web Audio API para música e efeitos

## 🎨 Design

- Visual estilo voxel/Minecraft
- Pós-processamento com bloom
- Iluminação dinâmica e sombras
- Ciclo dia/noite
- Ambient Occlusion

## 📝 Notas

- O jogo roda 100% no navegador
- Não requer instalação
- Funciona em PC e mobile
- Mundo infinito gerado proceduralmente
